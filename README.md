# GB Balancing Market Volatility & Spike Risk Modelling
GB electricity balancing market volatility and spike risk modelling using GARCH, EVT, and real Elexon data.



This project analyses the GB electricity balancing market using real Elexon balancing market and weather data.

The analysis focuses on:
- price volatility
- spike risk
- GARCH modelling
- Extreme Value Theory (EVT)
- stress regime analysis

## Tools
- Python
- Pandas
- NumPy
- Statsmodels
- ARCH
- Scikit-learn
- Matplotlib

## Key Features
- GARCH and Student-t volatility modelling
- Value at Risk (VaR) and Expected Shortfall
- Logistic regression spike classification
- EVT tail-risk estimation
- Regime-based volatility diagnostics
